## The Business to Code Bridge (Added: May 29, 2026)
This section serves as the permanent memory for how the **Business Dashboard** interacts with the **Sovereign Developer Studio (Aura Codes)**.

### Separation of Concerns
* **Aura Codes (The Engine Room):** The Sovereign Developer Studio. This is where all source code is written, managed, built, and hosted. No coding happens in the Business OS.
* **Master Dashboard (The Driver's Seat):** The operational UI for the business owner. They use this to manage staff, check GPS fleets, upload plans, and view analytics. It simply *consumes* the live URLs hosted by Aura Codes.

### UI Routing & Navigation
* **System Operations (Internal Inline Panels):** Settings like `Business Blueprint`, `POS Settings`, and `Pebble Settings` live natively inside the Master Dashboard (Port 5170). Clicking them triggers a smooth sliding animation to reveal the settings panel without opening new tabs.
